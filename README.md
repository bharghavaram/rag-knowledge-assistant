# RAG Knowledge Assistant (Open-Source)

A fully free, open-source Retrieval Augmented Generation (RAG) project built without paid APIs or local GPU requirements. This project runs using embeddings, Chroma vector DB, and open-source LLMs via Google Colab.

## Features
- PDF Document Ingestion
- Text Chunking
- Embedding Generation
- Vector Store Search
- RAG Query + Citations
- Evaluation Notebooks
- Open-Source Stack

## Stack Used
- Python
- LlamaIndex
- Sentence Transformers
- ChromaDB
- Streamlit (UI)
- Google Colab (Compute)
